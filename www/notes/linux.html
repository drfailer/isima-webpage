Shell scripting

Variables

Basic variables

a="something"
b=$(ps aux)
lit i = 0 # int

Declare an array

Here is an array of strings:

#+endsrc sh declare -a array=("hello" "world" ) #+endsrc

Conditions

Basic Conditions

if [[ $a == 0 ]]; then
echo "a is null"
fi

# or
if [ $a -eq 0 ]; then
echo "a is null"
fi

Check if file exists

if [ -f /path/to/file ]; then
    echo "the file exists!"
fi

Les Strings

if [[ $1 == "Hello" ]]; then
    echo "World!"
fi

Loops

Basic

for i in /etc/rc.*; do
    echo $i
done

C-like

for ((i = 0 ; i < 100 ; i++)); do
    echo $i
done

Ranges

for i in {1..5}; do
    echo "Welcome $i"
done

# step size:
for i in {5..50..5}; do
    echo "Welcome $i"
done

While

while true; do
    echo "hello"
done

Functions

my_fun() {
    echo "hello world"
}

Arguments

syntax utility
:------- :-------------------------------------
$# Number of arguments
$* All arguments (as single value)
$@ All arguments (as separated strings)
$1 First argument ($2, $3, …)
$_ Last argument of previous command

Some usefull command

Sed

Replace all hello by hello world. 

sed 's/hello/hello world/g'
Regex in sed
REGEX FUNCTION
$ end of line
^ first non blank char
. any char
+ one or any
* 0 or any
[abc] a or b or c
.{3} succession of 3 chars
() capture group
\1 reference to the first capture group

Printf

printf "%s\n" "hello"

Peut être utilisé pour séparer les éléments d'un tableau avec un caractère spécial: 

printf "%s\n" "${array[@]}"

Pacman

COMMAND UTILITY
sudo pacman -S pkg package installation
sudo pacman -Syu update all pkgs
sudo Syy double check
sudo pacman -Sw pkg download rather than install
pacman -Ss regx searching for program
sudo pacman -Rs pkg remove program and dependencies
sudo pacman -Rns pkg remove program, dep, and dotfiles
pacman -Q list all package installed
pacman -Qe list all program manually installed
pacman -Qn list all programs installed from the main repo
pacman -Qm list all programs installed from the aur
pacman -Qdt list dependencies
sudo pacman -Sc remove old packages from cash
sudo pacman -U pkg downgrade pkg

Arch Maintenance

Searching for errors

systemctl --failed
sudo journalctl -p 3 -xb

Cleaning

COMMAND USAGE
pacman -Sc and yay -Sc clean cache (and c to full clean)
pacman -Qtdq check for orphan packages
yay -Yc clean unwanted dependencies
journalctl --vacuum-time=2weeks clean journal files (keep 2 weeks)
find / -xtype l -print find brocken symlinc
rmlint

Utility program to install that removes duplicates files. Run rmlint as root and then run ./rmlint.sh (which is created after running the rmlint command). All the files that will be removed are listed in rmlint.json (file created in the current directory).

Reflector

Some reflector examples: 

sudo reflector -c France -a -6 -p https --sort rate --save /etc/pacman.d/mirrorlist --download-timeout=60
sudo reflector --verbose --connection-timeout 5 --cache-timeout 5 --latest 10 --sort rate -f 10 --save /etc/pacman.d/mirrorlist
sudo reflector --verbose -l 200 -p http --sort rate --save /etc/pacman.d/mirrorlist
