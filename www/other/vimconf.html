Installation via git

Les plugins sont dans le répertoir autotload/plugged cependant ce sont des répertoirs git donc il faut utiliser la commande git submodule add https://github.com/drfailer/simple-vim.

Sinon il faut installer les pluggins manuellements. Pour ce faire, il faut ouvrir vim et exécuter la commande :PlugInstall.
Cette commande va automatiquement télécharger et installer les pluggins.

Installation via le zip

Il suffit juste d'utiliser le script d'installation fourni et de confirmer l'emplacement de $HOME (il faut juste appuyer sur entrer pour garder la valeur par défaut).
